vue-qsType‑safe, reactive URL query params for Vue

现代化的 URL 查询参数状态管理，具备完整类型安全、错误处理和清晰架构。

快速开始
API

类型安全与现代化

完整的 TypeScript 支持，具备全面错误处理和清晰架构，使用现代 Vue 3 模式。

灵活的适配器

支持 History API、Vue Router、哈希路由（类似 VueUse）或自定义适配器。通过依赖注入实现关注点清晰分离。

丰富的序列化器

内置字符串、数字、布尔值、日期、JSON、数组和枚举编解码器，支持自定义序列化。

积极开发提醒

该库目前正在积极开发中，尚未准备好用于生产环境。API 可能在版本之间发生重大变化。请谨慎使用，并预期会有破坏性更改。
